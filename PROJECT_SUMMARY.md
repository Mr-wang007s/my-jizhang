## 📦 项目结构

```
my-jizhang/
├── src/
│   ├── actions/          # Redux 异步操作
│   ├── reducers/         # Redux 状态管理
│   ├── store/           # Redux Store
│   ├── components/      # 可复用组件
│   ├── pages/           # 4 个页面 (首页/添加/统计/设置)
│   ├── constants/       # 常量和类型定义
│   ├── services/        # 本地存储服务
│   ├── utils/           # 工具函数 (日期/计算/通用)
│   ├── styles/          # 全局 iOS 风格样式
│   └── assets/          # 静态资源
├── config/              # Taro 构建配置
├── package.json
└── README.md            # 完整文档
```

由于 Taro 需要实际图片文件作为 Tab Bar 图标，当前已注释 Tab Bar 配置。

**解决方案**:
1. 在 `src/assets/images/` 添加 6 个 40x40px 图标
2. 取消 `src/app.config.ts` 中 `tabBar` 配置的注释

### 2. 日期选择器
当前使用快捷日期选择 (今天/昨天/前天)。

**增强方案**:
