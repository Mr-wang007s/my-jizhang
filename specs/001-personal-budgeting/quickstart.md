# Quickstart: 个人记账与消费趋势统计

**Feature**: 001-personal-budgeting

本快速上手指南从用户视角描述如何使用个人记账功能记录收支并查看消费趋势，不涉及具体技术实现细节。

## 1. 初次使用

1. 打开应用后进入首页，可看到当月的收支概览区域和最近几笔记录列表（若暂无记录则显示空状态提示）。
2. 确认默认时间范围为「本月」，后续可按需切换。

## 2. 记录一笔收支

1. 在首页点击「记一笔」或类似入口，进入记账页面。
2. 输入金额，选择类型（收入或支出）。
3. 选择日期（默认为今天，可调整）。
4. 选择分类（如工资、餐饮、交通等），必要时添加备注。
5. 点击保存，返回后应在列表顶部看到刚才新增的记录，同时首页概览中的汇总金额会更新。

## 3. 查看账单列表与汇总

1. 在首页或导航中进入「账单」或「明细」页面。
2. 在页面顶部选择时间范围：如本月、本周、近 30 天或自定义日期。
3. 列表区域显示该时间范围内的所有记录，可按时间倒序查看。
4. 页面上方或底部展示该时间段的总收入、总支出和结余，随筛选条件改变自动更新。

## 4. 查看消费趋势和统计

1. 打开「统计」或「趋势」页面。
2. 在时间筛选区域选择想要分析的时间范围（如最近 3 个月、今年）。
3. 在「趋势」视图中查看按月份汇总的支出变化，例如折线图或柱状图形式。
4. 在「分类分布」视图中查看按分类统计的支出占比，识别出支出占比最高的几个分类。

## 5. 管理已有记录

1. 在账单列表中点击某条记录，可进入详情或编辑页面。
2. 修改金额、日期、分类或备注后保存，返回时列表和汇总应更新为最新结果。
3. 如需删除记录，可在详情页面选择删除操作，删除后该记录不再出现在列表和统计中。

## 6. 使用建议

- 养成每日记账习惯，可以在每天固定时间集中记录当日收支。
- 定期（如每周或每月）查看趋势和分类统计，发现不必要的支出并做出调整。
- 若发现统计结果与预期不符，可检查是否有漏记或误记的账目并及时更正。
