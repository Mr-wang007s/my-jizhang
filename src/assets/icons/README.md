# Tab Bar Icons

Please add the following icon files (40x40px PNG):

- home.png - Home icon (gray)
- home-active.png - Home icon (blue #3B82F6)
- list.png - List icon (gray)
- list-active.png - List icon (blue #3B82F6)
- chart.png - Chart/trend icon (gray)
- chart-active.png - Chart icon (blue #3B82F6)
- settings.png - Settings/gear icon (gray)
- settings-active.png - Settings icon (blue #3B82F6)

For development, you can use placeholder icons from:
- Iconfont (iconfont.cn)
- IconPark (iconpark.oceanengine.com)
- Or design your own in Figma/Sketch
